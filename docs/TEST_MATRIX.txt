COMPILE / TOOLCHAIN SURFACE
---------------------------

    gcc_strict_c17
    clang_strict_c17
    gcc_asan_ubsan
    win32_conditional_api_surface
    msvc_strict_c17

The Win32 surface test compiles every `_WIN32` conditional source path against
minimal API declarations with Clang warnings-as-errors. It catches conditional
compilation and API type mistakes, but it does not replace the Windows SDK,
linking, execution, ABI validation, or the MSVC row.
